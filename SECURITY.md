# Security Policy

SoftLoaf Trichrome is pre-1.0 software. Security-sensitive issues may include
crashes or memory-safety bugs triggered by untrusted image files, archive or
package signing issues, and unsafe update or distribution workflows.

Please do not disclose exploitable issues publicly before the maintainer has a
reasonable chance to investigate. Open a private advisory on GitHub if
available, or contact the project owner directly.

When reporting, include:

- affected commit or release;
- operating system and architecture;
- file type and camera model if the issue involves RAW decoding;
- minimal reproduction steps;
- whether the sample file can be shared.

# v0.2.0-beta.2

SoftLoaf Trichrome beta 2 tightens the export surface so the app only exposes
formats and workflows that are implemented end to end.

This is still a beta release. It is suitable for broader testing of trichrome
import, preview, TIFF export, color-managed output, and guarded RAW workflows,
but it is not a full camera compatibility or RAW color-correctness guarantee.

## Changes

- Removed the unfinished DNG-family export option from the export window.
- Added a backend guard so any stale or scripted DNG export request fails
  immediately with a clear "choose TIFF" message instead of decoding full RAW
  frames and failing at write time.
- Allowed export to start while background preview-cache processing is still
  active.
- Paused background frame preview processing when export starts so large RAW
  exports do not compete with cache warmup.
- Kept TIFF export as the implemented output path with 8-bit/16-bit selection,
  ICC profiles, and named color-space targets.
- Corrected README, architecture notes, todo status, and older release notes so
  they no longer describe DNG output as currently supported.
- Corrected the documented preview-cache long edge from 4096 px to the actual
  2048 px desktop cache behavior.
- Replaced stale extraction notes about JSON reports and metadata-policy stubs
  with the implemented project metadata and file identity probes.

## Known Limitations

- DNG import remains supported through LibRaw where LibRaw can open the source,
  but DNG export is not implemented in this release.
- RAW classification/provenance boundaries are protected, but this release does
  not claim full camera-family coverage.
- RAW color correctness is not RawTherapee parity; remaining differences are
  documented in `docs/raw_correctness_report.md`.
- Windows packages may still trigger SmartScreen if Authenticode signing is not
  configured for this release.

  notarization, and macOS 15.0 `minos` compatibility checks.
- Windows release publishes both installer and portable zip packages with Qt,
  OpenCV, LibRaw, Little CMS, and runtime dependencies.

## 更新内容

- 支持三张 R/G/B 分色照片导入、分组和合成预览。
- 支持 LibRaw 读取常见 RAW 格式。
- 支持 TIFF、JPEG、PNG 输入。
- 支持预览缓存，加快重复操作。
- 支持导出合成后的 trichrome 图像。

## 注意事项

- 当前版本是 alpha 测试版，不建议用于正式归档生产流程。
- Windows 安装包和便携包暂未签名，首次运行可能出现 SmartScreen 提示。
- RAW 相机兼容性仍在扩展和验证中。
- ICC / 色彩管理仍在完善，ACES AP0 和 ACEScg/AP1 路径目前属于早期支持。

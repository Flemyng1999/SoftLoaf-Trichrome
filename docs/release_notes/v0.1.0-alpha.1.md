# SoftLoaf Trichrome v0.1.0-alpha.1

This is an alpha release for early testing of the standalone SoftLoaf Trichrome
desktop app.

## Supported Platform

- macOS 15.0 or newer
- Windows 10 or Windows 11, x64 tester build

The Windows installer and portable zip are unsigned in this alpha. Windows
SmartScreen may warn on first launch or installation.

## What's Included

- Three-image trichrome import and alignment workflow.
- LibRaw-backed RAW input for common camera formats.
- TIFF, JPEG, and PNG input through OpenCV.
- Preview cache for faster repeat interactions.
- Export path for composed trichrome output.

## Known Limitations

- RAW camera compatibility is still being expanded and verified.
- ICC handling is still conservative. ACES AP0 and ACEScg/AP1 export paths are
  present, but profile coverage and metadata verification are not complete.
- Windows packages are not yet signed for general release.
- This alpha should be treated as test software, not archival production
  tooling.

## Packaging Notes

The macOS artifact must be built on a macOS 15 release environment with
`CMAKE_OSX_DEPLOYMENT_TARGET=15.0`. Do not publish local DMGs whose bundled
Qt/OpenCV/LibRaw dependencies report `LC_BUILD_VERSION minos` newer than 15.0.

The Windows x64 artifacts are built on a clean Windows runner with vcpkg,
`windeployqt`, and NSIS. Treat them as unsigned tester builds until Authenticode
code signing is configured.

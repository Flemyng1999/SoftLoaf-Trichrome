## 更新内容

- 修复 macOS 包内 OpenCV 动态库仍指向 Homebrew 路径，导致用户机器启动即崩溃的问题。
- 打包阶段会在签名前修复 bundle 内动态库的 install name、rpath 和本地绝对路径依赖。
- macOS 兼容性检查新增动态库解析校验，避免再次发布依赖本机 `/opt/homebrew` 的包。
- 改善 Windows/macOS 路径显示和导入路径处理，减少中文路径、空格路径和平台路径格式造成的可用性问题。
- 精简导出界面，并继续支持 TIFF 8-bit / 16-bit 选择。该版本仍暴露过未完成的
  DNG-family 输出选项；后续版本已移除，直到真正的 DNG writer 落地。

## 注意事项

- 当前版本仍是 alpha 测试版，不建议用于正式归档生产流程。
- Windows 安装包和便携包如未配置 Authenticode 证书，首次运行可能出现 SmartScreen 提示。
- RAW 相机兼容性仍在扩展和验证中。
- ICC / 色彩管理仍在完善，ACES AP0 和 ACEScg/AP1 路径目前属于早期支持。

## Changes

- Rebuilt export color management around centralized colourspace definitions inspired by `colour-science/colour`.
- Added Little CMS generated ICC profiles for all named TIFF-family export targets, including ACES AP0 Linear and ACEScg/AP1 Linear.
- Kept ACES AP0, ACEScg/AP1, Rec.2020, and ProPhoto export spaces explicitly linear.
- Added matrix regression tests against colour-science reference values for ACES AP0, ACEScg/AP1, Rec.2020, and ProPhoto.
- Added ICC generation and 16-bit ACEScg TIFF round-trip tests.
- Documented the current color-management pipeline and the remaining RAW camera-profile boundary.

## Known Limitations

- RAW camera compatibility is still being expanded and verified.
- DCP/profile lookup, camera-specific matrices, white-balance policy, and RawTherapee-informed fixture coverage remain future RAW correctness work.
- Windows packages may still trigger SmartScreen if Authenticode signing is not configured for this release.
